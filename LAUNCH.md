Post draft:

I built Maintainer Signal, a small GitHub Action and CLI for open-source maintainers.

It generates weekly reports for issue triage, project health, stale issues, label suggestions, release-note candidates, and optional OpenAI-powered maintainer briefs. It is read-only by default and has zero runtime dependencies.

Repo: https://github.com/Luke2program/maintainer-signal

Would love feedback from maintainers who want a simple CI-native digest instead of another dashboard.
